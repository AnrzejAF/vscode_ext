# Anrzej Color Themes

A VS Code theme extension with curated color themes by Anrzej.

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "Anrzej Color Themes"
4. Click Install

## Features

- Black Orange theme included
- Curated color palette
- Optimized for readability
- Support for syntax highlighting
